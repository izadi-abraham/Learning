## How I work

1. Copy `TEMPLATE.md` to `problems/<number>-<slug>.md`.
2. Solve it in `problems/<number>-<slug>.ts` (or any language).
3. Add a line to the log below.
4. Write down what the pattern taught me in `patterns/<pattern>.md`.
